# monster-dungeon

## What it does?
Click anywhere on the canvas to place a monster. The program uses Euclidean distance collision detection to ensure they do not overlap, retrying up to 870 times to find a position before the program declares that the dungeon is full.

## Used Concepts:
- Collision detection algorithm
- Randomized placement
- Event-driven programing


## You can use this program by making sure python is installed, and using any python editor. Run the file and click to randomly place the "monsters"
